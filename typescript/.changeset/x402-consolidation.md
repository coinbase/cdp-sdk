Add new `@coinbase/x402`, `@coinbase/x402-express`, `@coinbase/x402-hono`, and `@coinbase/x402-next` packages — Coinbase CDP-opinionated wrappers for the x402 HTTP payment protocol. These packages provide CDP-authenticated facilitator clients, payment clients with EVM and Solana wallet support, server middleware for popular Node.js frameworks, spend controls (guardrails), and Bazaar resource discovery.
